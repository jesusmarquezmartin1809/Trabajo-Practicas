# Gestor de Prácticas HLANZ

Aplicación web desarrollada para el instituto :contentReference[oaicite:0]{index=0} con el objetivo de gestionar las prácticas del alumnado y facilitar la relación entre estudiantes, empresas y tutores.

El sistema permite administrar la asignación de alumnos a empresas, realizar el seguimiento de las prácticas y centralizar toda la información relacionada con la Formación en Centros de Trabajo (FCT).

---

##  Objetivos del proyecto

- Gestionar alumnos y empresas colaboradoras.
- Asignar estudiantes a empresas de prácticas.
- Facilitar el seguimiento por parte de tutores.
- Centralizar información y documentación.
- Mejorar la organización del proceso de prácticas.

---

##  Funcionalidades principales

###  Gestión de alumnos
- Alta, edición y consulta de estudiantes.
- Estado de las prácticas.
- Información académica básica.

### Gestión de empresas
- Registro de empresas colaboradoras.
- Datos de contacto.
- Historial de alumnos asignados.

### Asignación de prácticas
- Relación alumno ↔ empresa.
- Control de disponibilidad de plazas.
- Seguimiento del estado de la asignación.

### 🧑‍💼 Panel de administración
- Gestión centralizada del sistema.
- Búsquedas y filtros.
- Visualización global del estado de prácticas.
